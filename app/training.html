Команда состоит из 6 игроков и у каждого есть своя позиция

  • Либеро
  • Связующий
  • Центральный блокирующий
  • Диагональный
  • Доигровщик
  • Подающий

Уникальность позиций

У каждого игрока есть своя позиция (для которой нужны различные умения) и номер на площадке

Роль позиции

У каждого игрока есть своя роль и цели которые ей принадлежат

Умения для позиций

Каждый игрок должен умениями для своих позиций, например:

связующий должен уметь хорошо давать пасы,

диагональный должен уметь забивать мяч

из определённой позиции

Позиция Умение
подающий подавать обычную подачу, по возможности планер или силовую подачу
либеро принимать атаки соперника
связующий давать удобные для удара пасы на атакующих игроков

диагональный атаковать по диагонали
